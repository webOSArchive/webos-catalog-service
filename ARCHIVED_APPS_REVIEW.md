## Status Options

- **Active** - IPK is available in the museum
- **Missing** - IPK is wanted (will appear in wanted.txt)
- **Newer** - Post-freeze submission with IPK available
- **Archived** - Historical reference only (not served, not wanted)

## How to Change Status

1. Go to `/admin/apps.php`
2. Filter by status "archived" or search by app ID/title
3. Click **Edit** on the app
4. Change the **Status** dropdown to the appropriate value
5. Click **Save Changes**

## Notes

- AccuWeather apps may have patched versions already in the catalog
- Some games (TRANSFORMERS, Slot Machine) might be good candidates for the wanted list
- After changing status to "missing", regenerate wanted lists via Dashboard → Utilities → Generate Wanted Lists
